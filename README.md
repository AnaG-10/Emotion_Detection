This research project compares emotion-recognition performance between fuzzy logic-based models and deep learning CNN models. We evaluate EfficientNet, ResNet, and a Fuzzy Emotion Classifier, trained on two different datasets, giving a total of six trained models.

The goal is to analyze:

- Performance difference between Fuzzy and Neural Network approaches

- Comparison between EfficientNet vs ResNet vs Fuzzy

- How dataset quality affects accuracy

- Which model generalizes best during inference
